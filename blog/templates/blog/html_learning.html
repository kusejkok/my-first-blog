HTML commands I learned:

  1. The tag head and title define what is shown in the top bar, not on the page itself.
  2. The tag body defines what is shown on the page.
  3. The tags h1, h2, ..., h6 define the headings.
  4. The tag p stands for paragraph.
  5. The tag a is used for links, e.g., the link to this tutorial: This is a link
  6. The tag ol stands for ordered list, ul for unordered list. li are the list elements.
  7. Line breaks are made with
    a br tag. This is called an empty element, no end tag needed.
  8. Horizontal lines can be included using hr
  9. The tags b, strong display items in bold, but strong means important!
  10. The tags i, em displays items in italic, but em means important!
  11. The tag small can make some words smaller than others.
  12. The tag mark marks words.
  13. The tag del can cross out words.
  14. The tag ins can underline words.
  15. The tag sub can subscripts text and sup can superscript words.

The style parameters can be used to define color, font, size

Font sizes can be specified for single paragraphs using font-size:20px

Background colors can be defined in the style part as well!

Fonts can be defined in the style part as well!

The style attribute text-align: center can center elements.

With q we can make quotations.

We can make a box around elements!: style = "border:2px solid Violet; padding: 30px;"

Good to know: 

With  pre  you can add preformatted text.

HTML is NOT case sensitive.
