Chapter 1: Python Basics

General

  • Variable names in Python can contain letters, numbers and _, but must start with a letter
  • Variables are assigned with =
  • Python uses the common operators: +, -, *, /. // gives back the integer part of the division, % the remainder. ** powers.

Functions learned

  • str(), int(), float() will evaluate the input as string, integer and float.
  • print() displays the argument.
  • len() gives the length of a string.
  • input() asks the user for input.

Chapter 2: Flow control

General

  • Boolean values in Python are "True" and "False".
  • The three Boolean operators are "and", "or" and "not".
  • The six comparison operators are: ==, !=, <, <=, >, >=
  • A Python program can be exited with ctr C; or sys.exit() in the code.
  • The function range() can take 1, 2 or 3 arguments. (from, to by)
  • Importing packages in Python needs "import ..." at the beginning, e.g., import random
  • Function from an imprted module are called with module.function(), e.g., random.randint(5)

Flow control statements

  • if condition: do..., elif, else
  • while condition: do..., you can break with "break" or continue with "continue"
  • for i in range(5): condition...
  • input() asks the user for input.
